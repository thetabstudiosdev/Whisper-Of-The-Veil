# ThetaB Studios

## Official Documentation Portal

Welcome to the official GitHub documentation repository for **Whisper of the Veil**, an indie dark fantasy RPG developed by ThetaB Studios.

This repository contains official information about the project, studio, legal policies, and future development documentation.

---

## 🎮 About the Game

**Whisper of the Veil** is a dark fantasy role-playing game focused on narrative storytelling, exploration, and atmospheric worldbuilding.

The game is being developed by **ThetaB Studios** as an independent creative project.
